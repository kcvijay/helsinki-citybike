## Run in your machine

- Step 1  
  Go to GitHub repo and fork it.

- Step 2
  From your forked repo

```
git clone <forked-repository-url>
```

- Step 3  
  In project directory, navigate to API folder

```
cd API
```

- Step 4
  Run backend server and database

````

npm run start
````
