# HELSINKI CITY BIKE APP (MERN TYPESCRIPT)

## Browse the customer journeys between 05.2021 - 07.2021

## Tech Stack

- React
- TypeScript
- Node
- Express
- MongoDB with Mongoose
- React-Redux Toolkit
- UI / Styling:
  - TailwindCSS, CSS3, Material UI, Google Icons

## Other NPM Packages

- Nodemon
- Axios
- React Router
- Toastify
